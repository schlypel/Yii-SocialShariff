Yii Extension - Social Shariff
=============================

Installation
------------

    1. Copy the extracted files in protected/extensions

Basic usage
-----------
     $this->widget('ext.socialShariff.CoinSliderWidget', array(
		'theme' 		=> 'white',
		'orientation' 	=> 'vertical'
	));
